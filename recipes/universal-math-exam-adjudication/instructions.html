Math Expression Formatting

  • Questions use LaTeX formatting for mathematical expressions.
  • In-line math expressions should be surrounded by single dollar signs: $expression$
  • If you need an actual dollar sign to appear in text, escape it with a backslash: \$
  • Some questions may use \( and \) instead of dollar signs for math. These can (optionally) be replaced with $ for consistency, which has the same effect.
  • If extensive changes to mathematical expressions are needed, please consult these resources:
      LaTeX syntax for math operations are described in this cheat sheet.
